SEJARAH JAVASCRIPT

JavaScript dibuat tahun 1995 oleh Brendan Eich di Netscape untuk menambah interaktivitas web. Awalnya bernama Mocha ‚Üí LiveScript ‚Üí JavaScript. Tahun 1997 distandardisasi jadi ECMAScript agar seragam di semua browser.

Versi penting: ES3 (1999) membuatnya populer, ES5 (2009) menambahkan strict mode & JSON, lalu ES6 (2015) membawa perubahan besar seperti let/const, class, arrow function, module, dan promise. Setelah itu rilis tahunan terus menambah fitur modern seperti async/await dan optional chaining.

Kini JavaScript dipakai di front-end, back-end (Node.js), mobile, desktop, IoT, hingga AI, menjadikannya bahasa pemrograman paling populer di dunia.
